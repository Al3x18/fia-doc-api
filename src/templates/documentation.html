02

Security

Authentication

Protected endpoints require a server-configured API key in the X-API-Key request header. The expected value is stored by the service in FIA_DOCS_API_KEY.

!
Keep credentials out of browser code.

Do not embed the API key in a public website. Send protected requests from a trusted backend or controlled client. This documentation page contains no key, JavaScript, or live API console.

REQUEST HEADER 
X-API-Key: $FIA_DOCS_API_KEY

03

Recommended sequence

From season to document in four steps.

  1. 1
    Discover the season GET /get-seasons-available
  2. 2
    Resolve the event name GET /get-gp-available?season=2025
  3. 3
    Retrieve document metadata GET /fia-documents?season=SEASON 2025&event=Abu Dhabi Grand Prix
  4. 4
    Stream the selected PDF GET /download-fia-doc?url={fia_document_url}

GET /get-championships-available
PROTECTED

Returns FIA championships available for a season. A year-only value is normalized to SEASON YYYY.

ParameterTypeRequiredDescription
seasonstringNoFull season label or four-digit year.
REQUEST PATH

GET /get-gp-available
PROTECTED

Returns event names accepted by the document endpoint for the selected season.

ParameterTypeRequiredDescription

Runs the FIA scraper and returns normalized document metadata for one event. When event is omitted, the most recent available event is used.

GET /download-fia-doc
PROTECTED

Streams a PDF from an approved fia.com HTTPS URL. Redirect destinations are validated and external hosts are rejected.

ParameterTypeRequiredDescription
urlURLYesFIA document URL returned by /fia-documents.
200 application/pdf Content-Disposition: attachment
GET /track-image
PUBLIC

Returns a matching circuit PNG from the local track asset library. Matching is case-insensitive and separator tolerant.

ParameterTypeRequiredDescription

Error model

400Bad request

Missing parameter, invalid document host, or malformed input.

401Key required

The X-API-Key header was not provided.

403Key rejected

The provided API key does not match the server value.

404Asset not found

No matching track image exists.

500Upstream failure

FIA navigation, parsing, or download failed.

503Auth unavailable

FIA_DOCS_API_KEY is not configured on the server.

07

Runtime behavior

Operational notes

Upstream dependency

Discovery and document endpoints depend on the current FIA website structure and network availability.

Responsible usage

Cache stable results where appropriate and avoid unnecessary repeated scraping of the same event.

Season fallback

If the requested current season is unavailable, document retrieval falls back to the previous year.

Static documentation

This page is server-rendered HTML. It loads one local stylesheet and makes no API or third-party requests.
